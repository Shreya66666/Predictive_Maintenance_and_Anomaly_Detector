A comprehensive machine learning pipeline engineered to predict mechanical failure anomalies using high-frequency IoT sensor data. This project leverages Pandas for extensive exploratory data analysis (EDA) and feature engineering, generating time-series aggregations and rolling averages to capture equipment degradation trends. Recognizing the severe class imbalance inherent in failure detection (where normal operations vastly outnumber failures), the pipeline integrates the Synthetic Minority Over-sampling Technique (SMOTE) to prevent model bias. The predictive engine is powered by an optimized XGBoost gradient boosting classifier. Evaluated against stringent metrics, the model successfully identifies equipment failures with high precision, achieving an exceptional AUC-ROC score. This system demonstrates end-to-end data science proficiency, from raw data wrangling and algorithmic balancing to predictive deployment.
